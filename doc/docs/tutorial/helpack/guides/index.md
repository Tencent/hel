# Helpack 三种模式

Helpack 支持三种运行模式，从简单到复杂，满足不同的使用场景。

## 模式概览

| 特性 | 模式 1 (Simple) | 模式 2 (Local) | 模式 3 (完整) |
|------|----------------|----------------|---------------|
| 数据库 | ❌ 不需要 | ❌ 不需要 | ✅ 需要 MySQL |
| Redis | ❌ 不需要 | ❌ 不需要 | ✅ 需要 Redis |
| 版本切换 | ✅ 支持 | ✅ 支持 | ✅ 支持 |
| 数据持久化 | ❌ 不支持 | ❌ 不支持 | ✅ 支持 |
| 用户管理 | ❌ 不支持 | ❌ 不支持 | ✅ 支持 |
| 完整 API | ❌ 部分 | ❌ 部分 | ✅ 完整 |
| 适用场景 | 快速体验 | 本地开发 | 生产环境 |

## 模式选择

- **[Simple 模式](./mode1-simple.md)** - 最简单的模式，快速体验 Helpack
- **[Local 模式](./mode2-local.md)** - 本地开发模式
- **[完整模式](./mode3-full.md)** - 完整生产模式

根据你的需求选择合适的模式开始使用。
